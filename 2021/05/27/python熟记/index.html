字符串

str.split(sep='-', maxsplit=0)

split()拆分字符串；sep表示为分隔符，默认为空格；maxsplit表示最多将str分隔为maxsplit+1个子字符串。

去除空格

strip()方法，去除字符串开头或者结尾的空格

replace()方法，可以去除全部空格

循环

for i, j in product(A, B):

返回A列表和B列表形成的笛卡尔积元素；product(A, repeat=4) means the same as product(A, A, A, A)；产生二维网格：for i, j in product(range(10), repeat=2):。
